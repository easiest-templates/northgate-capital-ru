# Northgate Capital

Премиальный шаблон лендинга для инвестиционной компании с элегантным дизайном и русской локализацией.

![Preview](https://cdn.poehali.dev/templates/northgate-capital-ru/preview-ru.jpg)

## Особенности

- Современный адаптивный дизайн
- Премиальная типографика (Playfair Display + Inter)
- Анимации и плавные переходы
- Полная русская локализация
- Готовые секции: герой, о компании, портфель, команда, контакты

## Технологии

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Radix UI
- Lucide Icons

## Установка

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```
