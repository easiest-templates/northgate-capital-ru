# Northgate Capital — Инвестиционная компания

![Preview](https://cdn.poehali.dev/templates/northgate-capital-ru/preview-ru.jpg)

Готовый шаблон сайта для инвестиционной компании или фонда. Подойдет для тех, кто хочет представить свою компанию солидно и профессионально. Темная цветовая схема создает ощущение премиальности и надежности.

Шаблон включает все необходимые разделы: главный экран с эффектным заголовком, блок статистики с анимированными числами, информация о компании и инвестиционном подходе, портфолио проектов, кейсы с результатами, раздел ESG (экология и социальная ответственность), команда с фотографиями и описаниями, блог с аналитикой и форма обратной связи.

Все тексты уже на русском языке. Дизайн адаптивный — отлично смотрится и на компьютере, и на телефоне. При прокрутке страницы элементы плавно появляются, что создает приятное впечатление. Числа в блоке статистики анимированно отсчитываются при появлении на экране.

Чтобы использовать шаблон, просто замените тексты и фотографии на свои. Структура понятная — каждый раздел находится в отдельном файле, поэтому легко найти нужное место для редактирования.
